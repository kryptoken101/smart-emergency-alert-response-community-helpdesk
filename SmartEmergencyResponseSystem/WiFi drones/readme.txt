Version 1.

Date: 04/13/2014
Functionality: Read GPS data from a drone using the serial communication, and send the data to Google Earth to display.   

Main.m is where the program starts.  
Readserialport.m reads position data from drone.
Readtextfile.m reads position data from a textfile for testing popuse.
Googleearthplot.m displays position data on google earth in real time.
